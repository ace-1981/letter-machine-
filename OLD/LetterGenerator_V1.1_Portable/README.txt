מחולל מכתבים — Letter Generator (Portable)
==========================================

אין צורך בהתקנה. העתיקו את כל התיקייה למיקום כלשהו והפעילו app.exe בדאבל-קליק.

מבנה תיקייה
-----------
LetterGenerator_release/
  app.exe
  templates/
    תחשיב זכויות אישי.docx
    תחשיב זכויות אישי.json
  output/
  README.txt

קבצים שחובה לשמור ליד app.exe
------------------------------
- app.exe
- templates/תחשיב זכויות אישי.json  — מיפוי, תנאים, שם קובץ PDF, חתימה
- templates/תחשיב זכויות אישי.docx — טקסטים ועיצוב המכתב
- תיקיית output/ (נוצרת אוטומטית אם חסרה)
- README.txt

דרישות מערכת
------------
- Microsoft Word מותקן במחשב — נדרש להמרת DOCX ל-PDF.
- Windows 10 ומעלה.

עריכת תבנית והגדרות — בלי לבנות exe מחדש
-----------------------------------------
טקסטים ועיצוב:
  ערכו את קובץ ה-DOCX בתיקיית templates.
  לאחר שינוי — הפעילו Preview מחדש.

מיפוי, תנאים, שם קובץ PDF ושדה חתימה:
  ערכו את קובץ ה-JSON בתיקיית templates.
  לאחר שינוי — סגרו ופתחו מחדש את התוכנה (או Preview מחדש).

אין צורך לבנות app.exe מחדש עבור שינויי תבנית או הגדרות.

הפעלה
-----
1. שימו את כל הקבצים באותה תיקייה (כמבנה למעלה).
2. הפעילו app.exe בדאבל-קליק.
3. בחרו קובץ Excel (xlsx / xls) ותיקיית יעד (ברירת מחדל: output/).
4. בחרו סוג מכתב והפיקו או השתמשו בתצוגה מקדימה.

הערות
-----
- חתימה דיגיטלית אינטראקטיבית נתמכת ב-Adobe Acrobat / Foxit (לא ב-Chrome).
- אם חסר DOCX או JSON — תוצג הודעת שגיאה בהפעלה ולא תתרחש קריסה.

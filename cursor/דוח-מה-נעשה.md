## 2. מה נבנה

נוצרה תיקיית פרויקט `LetterGenerator/` עם מערכת Python מודולרית.

### 2.1 מבנה הקבצים

```

│   └── תחשיב זכויות אישי.docx   ← תבנית Word (נוצרה ל-POC)
├── samples/
│   ├── sample_data.xlsx          ← Excel לדוגמה (שורה אחת)
│   └── poc_output/               ← פלט POC
│       ├── 12345 כהן ישראל תחשיב זכויות אישי.docx
│       └── 12345 כהן ישראל תחשיב זכויות אישי.pdf
├── scripts/
│   ├── create_sample_excel.py    ← יצירת Excel לדוגמה
│   ├── create_template.py        ← יצירת תבנית DOCX
│   └── test_xls_support.py       ← בדיקת תמיכה ב-xls
└── src/
    ├── excel_reader.py           ← קריאת xlsx/xls + מיפוי עמודות
    ├── template_engine.py        ← החלפת משתנים ב-DOCX (docxtpl)
    ├── pdf_converter.py          ← המרה ל-PDF (Word COM / LibreOffice)
```

## 3. זרימת העבודה שמומשה (POC)

```
Excel (שורה 1)
    ↓
קריאת ערכים לפי עמודות C, E, F, H…S, T, R
    ↓
חישוב שדות (FULL_NAME = שם משפחה + שם פרטי)
    ↓
הערכת תנאים (הצגת מקטעים מותנים)
    ↓
validate — בדיקות לפני הפקה
    ↓
מילוי תבנית DOCX (docxtpl)
    ↓
המרה ל-PDF (Word COM — Word מותקן במחשב)
    ↓
הוספת שדה חתימה אינטראקטיבי /Sig (pyHanko)
    ↓
שמירה: "12345 כהן ישראל תחשיב זכויות אישי.pdf"
```

## 4. פירוט המודולים

### excel_reader.py
- קורא `.xlsx` (openpyxl) ו-`.xls` (xlrd)
- ממפה עמודות לפי אות עמודה (C, E, F, H…)
- מחשב שדות מורכבים (`FULL_NAME`)
- מעריך תנאים לוגיים (`L > 0`, `T == 'נקלט.ת'` וכו')

### template_engine.py
- ממלא את תבנית ה-Word במשתנים (`{{FULL_NAME}}`, `{{P}}`…)
- מוסיף תאריך אוטומטי (`{{TODAY}}`)
- בונה שם קובץ פלט לפי תבנית ב-JSON

### pdf_converter.py
- מחלקה מופרדת עם שני backends:
  - **WordComPdfConverter** — Microsoft Word (פעיל כרגע במחשב)
  - **LibreOfficePdfConverter** — LibreOffice headless (מוכן, לא מותקן)
- בוחר אוטומטית את הראשון הזמין

### signature_field.py
- מוסיף שדה AcroForm מסוג `/Sig` (לא מלבן גרפי)
- משתמש ב-**pyHanko**
- כולל פונקציית אימות פרוגרמטית (`verify_signature_field`)

### validator.py
בודק לפני כל הפקה:
- עמודות Excel חסרות
- משתנים חסרים בתבנית DOCX
- תקינות ביטויי תנאים
- תיקיית יעד קיימת וניתנת לכתיבה
- זמינות ממיר PDF

### letter_generator.py
- `generate_single_letter()` — הפקת מכתב בודד (POC)
- `generate_letters()` — הפקה מרובה; שגיאה בשורה **לא** עוצרת את הריצה
- שומר `errors_report.csv` בתיקיית היעד אם יש שגיאות

## 5. JSON טכני — מה יש ומה אין

קובץ: `LetterGenerator/templates/תחשיב זכויות אישי.json`

**יש (טכני בלבד):**
- מיפוי עמודות Excel
- שדות מחושבים
- תנאי הצגת מקטעים
- תבנית שם קובץ פלט
- הגדרות שדה חתימה
- רשימת שדות חובה ל-validate

**אין:**
- נוסחי טקסט (הועברו ל-DOCX)
- מפתח `texts` (בניגוד לטיוטה הישנה ב-`תחשיב זכויות אישי/תחשיב_זכויות_מעוצב.json`)

## 6. תוצאות הרצת POC

| פריט | ערך |
|------|-----|
| שורת Excel | 1 (כהן ישראל, קוד 12345) |
| שם PDF | `12345 כהן ישראל תחשיב זכויות אישי.pdf` |
| ממיר PDF | Word COM |
| שדה חתימה | `MemberSignature`, סוג `/Sig` |
| אימות אוטומטי | עבר — שדה חתימה אמיתי זוהה |

### אימות שדה חתימה (אוטומטי)

```json
{
  "found": true,
  "field_name": "MemberSignature",
  "field_type": "/Sig",
  "is_signature_field": true,
  "message": "Valid /Sig field detected"
}
```

### בדיקה ידנית (נדרשת ממך)

פתח את ה-PDF ב-Adobe Reader או Foxit → **Fill & Sign** → ודא שניתן לחתום בשדה `MemberSignature`.

## 1. מטרה כללית

בניית אפליקציית Windows portable ליצירת מכתבים מ-Excel + תבנית Word, בעברית RTL, עם תבניות חיצוניות הניתנות לעריכה בלי build מחדש.

## 3. שינויי V1.1 — מסך ראשי

### בחירת סוג פלט
- נוספו כפתורי רדיו: **PDF** (ברירת מחדל) / **Word / DOCX**
- **PDF:** נוצר PDF בלבד; DOCX זמני פנימי נמחק אחרי המרה
- **DOCX:** נוצר DOCX בלבד; ללא Word COM, ללא PDF
- **Preview:** פותח PDF או DOCX לפי הבחירה
- **Batch:** הלוג מציין סוג פלט; ספירת הצלחות לפי הפורמט

**קבצים:** `src/ui/main_window.py`, `src/ui/worker.py`, `src/output_format.py`, `src/letter_generator.py`

## 4. שינויי V1.1 — טבלת תחשיב

### לוגיקת שורות מותנית
| שורה | תנאי |
|------|------|
| תוספת בגין פטירה | L > 0 |
| קיזוז מענק עידוד עבודה | O > 0 |
| סה״כ אחרי קיזוז מענק | O > 0 |

ערכים ריקים / 0 / לא תקין → מטופלים כ-0 (`excel_reader.py`).

### עיצוב
- הוסר רקע מתחלף (zebra) בשורות רגילות
- רקע מיוחד לשורות סה״כ בלבד:
  - סה״כ בגין ותק (מספר שנים × סכום לשנת ותק)
  - סה״כ זכאות
  - סה״כ אחרי קיזוז מענק
- RTL, סכומים בפורמט `12,345 ₪`

**מימוש:** שורות נבנות ב-Python (`template_engine.py`, `docx_table_builder.py`) ומוזרקות לתבנית כ-subdoc.

## 5. שינויי V1.1 — הערות והבהרות

- כותרת **"הערות והבהרות"** מוסתרת כשאין אף הערה פעילה (`show_NOTES_SECTION`)
- כל הערה: כותרת מודגשת + גוף רגיל
- תנאים קיימים: פטירה (L), מענק (O), חבר חדש (T), חוב בנייה (R)

**קבצים:** `scripts/create_template_docx.py`, `excel_reader.py`

---

## 6. שינויי V1.1 — חתימה ותאריך

- מלבן חתימה רחב וגבוה יותר
- שדה תאריך: `תאריך: ______________________`
- JSON חתימה עודכן:
  - width: 300, height: 75
  - margin_bottom: 95, margin_left: 50
- שדה `MemberSignature` נשאר אינטראקטיבי ב-PDF (נבדק)

## 10. מה עדיין נדרש

| פריט | סטטוס |
|------|--------|
| **build חדש של app.exe** ל-V1.1 | נדרש |
| אישור ויזואלי ידני של המשתמש | ממתין |

```powershell
cd LetterGenerator
pip install -r requirements.txt
python scripts/build_release.py
```

לאחר build — להעתיק `templates/` מעודכנים ל-`LetterGenerator_release/templates/` (אם לא נעשה אוטומטית).

## 11. עקרונות שנשמרו

- אין הטמעת DOCX/JSON בתוך exe
- Portable — לא Installer
- עריכת תבנית/JSON אפשרית בלי rebuild
- Word נדרש רק להמרת PDF (לא ל-DOCX mode)

תיקייה זו מכילה **תיעוד ותכנון** שנוצר על ידי Cursor Agent — לא חלק מהתוכנה עצמה.

## מה נמצא כאן

| קובץ | תיאור |
|------|--------|
| `דוח-תכנון-ופיתוח.md` | דוח מלא: מה קיים, מה חסר, איך נבנה הפרויקט |
| `דוח-Visual-Template-Fidelity.md` | תיקון עיצוב תבנית Word |
| `דוח-בדיקות-לפני-GUI.md` | תוצאות 4 הבדיקות + GUI |
| `סטטוס-POC.md` | סטטוס עדכני לפי כיוון המעודכן |
| `README.md` | הקובץ הזה |

## הפרדה ברורה

```
מכונת מכתבים/
├── cursor/              ← של Cursor (דוחות, תכנון)
├── תחשיב זכויות אישי/  ← שלך (חומרי קלט)
├── מסמך אפיון...        ← שלך (אפיון)
└── LetterGenerator/     ← ייווצר — התוכנה (קוד + templates)
```

**כלל אצבע:** אם הקובץ בתוך `cursor/` — זה תיעוד שלי. אם מחוץ — או שלך או תוכנה.
